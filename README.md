# Caleb Lerch's Blog

Personal blog and portfolio site, built with [al-folio](https://github.com/alshedivat/al-folio) (Jekyll).

🔗 Live site: https://stomper4.github.io.com


(notes for myself)
## Local development

```bash
bundle install
bundle exec jekyll serve
```

## Deployment

Automatically deployed via GitHub Actions on push to `master`.

## Structure

- `_posts/` — blog posts
- `_projects/` — project pages
- `_news/` — news/announcements
- `_pages/` — static pages (about, CV, etc.)
